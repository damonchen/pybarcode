barcode
=======

## 缘由

若干年前，曾经做条码方面的工作，经常用C++来生成和识别条码，后来学习python，便想着
是否可以用python也来实现条码的生成，于是便有了这样一份代码。今天翻看历史代码，发现
还有这么一份代码趟在硬盘上，于是想着github上公开这份代码，可能更有价值一些。

## 条码实现

支持code11， code93， code39， code128， ean8，ean13，interleaved2of5，jsbn，jan13，
stardard2of5

## 依赖

该库基于PIL实现，因此需要  
`pip install PIL`


## 平台
当时在Windows上进行开发，因此也只测试过Windows上，从PIL上看，应该Linux之类的也是兼容的。

代码并没有仔细测试过，因此如果有问题，你可以自己解决，然后发一份patch给我，或者发邮件
我，跟我说哪些代码有问题，当然时间上可能会比你自己解决要晚一些。
